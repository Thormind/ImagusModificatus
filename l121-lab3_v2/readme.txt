Le programme permet de charger des images de format ".png". Il y en a une incluse dans le dossier "images".

Fonctionnalités supportées: 

- translation (on clique sur l'image et on la fait glisser)
- zoom (avec la roulette de la souris)
- sauvegarde (bouton UI ou CTRL+S)
- chargement (bouton UI)
- copier (CTRL+C)
- coller (CTRL+V) **lorsqu'une copie est collée, on peut choisir d'appliquer seulement certains réglages via l'UI
- défaire (bouton UI ou CTRL+Z)
- refaire (bouton UI ou CTRL+Y)

Le rapport final du projet se trouve dans le répertoire "documents". Il inclut plusieurs éléments de conception tels que
des diagrammes de classe et de séquence UML.
